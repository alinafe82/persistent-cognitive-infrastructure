# Trademark Policy

This repository is public source. The license grants rights to use, modify, and
distribute the code. It does not grant unrestricted rights to use project names,
logos, marks, or brand identity in a way that implies endorsement or official
status.

## Allowed Uses

You may use the project name to:

- identify the upstream project
- describe compatibility
- refer to unmodified source distributions
- publish truthful technical commentary
- attribute contributions

## Restricted Uses

Do not use project names, logos, or marks to:

- imply official endorsement
- market a hosted service as the official service without permission
- distribute modified versions as official releases
- impersonate maintainers
- confuse users about source, security, or support status

## Fork Naming

Forks should use names that are clearly distinguishable from the upstream
project unless they are temporary development forks on a public code hosting
platform.
